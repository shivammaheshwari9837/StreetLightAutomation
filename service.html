An intro to future changing Project

This project values-- Time, Money, Power, Human Effort. It uses smart sensors including- Co2 sensor, fog sensor, light intensity sensor, noise sensor, & GSM modules(specially Xbee). These sensors are controlled via efficient microcntrollers.

Once installed on lamps/lights, have minimal maintaince cost. The overall control is done by a master branch(a server) consits of Java essence, TomCat Server and the light weight MySql database.
